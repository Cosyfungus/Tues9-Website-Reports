Initial Design

ClassifyDriver - main function wrapper - check for command line arguments - create objects and manage function calls - manage output

TrainerDriver - main function wrapper - check for command line arguments - create objects and manage function calls - manage output

DocumentParser - parse document - pass in a filepath - validate file path - prune punctuation - case-insensitive when recognizing words - split document text into tokens - return parsed document object

Document - list to contain tokens - get words - get word count

Parser - pre-process - remove stop words from token list - read stop words from file - list of stop words

MLLibrary - normalization of documents (word counts) - weighs words on importance to classifying document - counts the number of occurrences of each word - list of weighted words and their values - hashtable for categories and their identifier(Letter)

Trainer extends MLLibrary (composition??) - train(trainingData) - placeholder for ML library calls WEKA

Classifier extends MLLibrary - classify documents into their categories
